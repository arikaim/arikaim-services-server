# Arikaim CMS python services server



#### Requirements 
  * [Arikaim CMS](https://github.com/arikaim/arikaim)


#### Installation

```sh

pip install arikaim-service-server

```

#### Usage

##### Run

```sh

    ./arikaim-server run

```

##### Create default config file

```sh

    ./arikaim-server config create

```
